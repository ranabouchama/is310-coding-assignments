### Rana Bouchama
### API used: https://poetrydb.org/index.html

<p>I selected the Poetry Database API as the Europeana API similarly works with novels or otherwise literature and thus decided to stay within a similar wheelhouse and look for an API that also mostly deals with literature data. The poetry database just so happened to be the only one available that was still usable as some others I was interested in were taken down. I was also interested in another database, however the site was in polish and thus there was a language barrier to using the API.</p>
