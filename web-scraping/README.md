# The Sailor Moon Crystal Wiki Scrape
### Rana Bouchama

## robots.txt Compliance

<p>The Sailor Moon Wiki does allow general user crawlers, with the only ones blocked being commercial crawlers, except for the AmazonAdBot https://sailormoon.fandom.com/robots.txt </p>

## Wiki Choice & Research Value

<p>I chose to go with the Sailor Moon Crystal wiki as it was an anime I grew up with and, as such, is near and dear to my heart. But more importantly, the information that may be useful to researchers is:

+ Character Network Analysis: I've included the data of character relations, and as such, you can create a web of which characters are related to whom and how
+ Character Affiliations: Building off of the social network that can be built, there is also relevant information on the organizations/groups the characters are affiliated with</p>
